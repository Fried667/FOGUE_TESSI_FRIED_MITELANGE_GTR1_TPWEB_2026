Introduction

Ce mini-site été créé dans le cardre du cours de tp programmation web.

Il permet de découvrir la creation de plusieurs pages HTML reliées entre elles.

Objectif du mini-site

L'objectif est d'apprendre a structurer un petit projet web avec plusieurs fichiers.

Pages disponibles

  • La page Acceuil présente le contexte général du mini-site.
  • La page Présentation contient les informations personnelles de l'étudiant
  • La page Contact contient les informations de contact de l'étudiant

J'apprends la programmation web parce qu'elle me permettra de créer des interfaces utiles pour les services réseau et les applications d'entreprise.
